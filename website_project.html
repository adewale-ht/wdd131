Project Subject

Subject: Nigerian Temples and Interactive AI Chat
Description: This website will feature information about various temples in Nigeria, including their history, images, and cultural importance. It will also include an interactive AI Chat page that demonstrates advanced JavaScript features and user engagement. The site will have at least four pages: a home page introducing the topic, a gallery page with optimized images and facts, a temples page with detailed information, and an AI Chat page. There will also be a references page for citations. The site will use a common navigation and theme, be fully responsive, and include dynamic JavaScript features such as weather info, interactive elements, and localStorage usage.
Reason: I chose this subject because I am passionate about sharing the rich heritage and spiritual significance of Nigerian temples, and I want to showcase my skills in web development by building an engaging AI Chat experience. My goal is to educate and inspire others about cultural value, while demonstrating proficiency in HTML, CSS, and JavaScript.

Project Requirements Overview

  • At least four pages with common navigation and theme
  • Significant, relevant content supporting the site’s purpose
  • Valid, standards-based HTML and CSS
  • Responsive design for mobile and desktop
  • Optimized images with lazy loading
  • HTML form meeting accessibility and validation standards
  • Dynamic JavaScript elements (multiple functions, DOM interaction, events, conditional branching, objects/arrays, template literals, localStorage)
  • Interactive AI Chat page with advanced features
  • No spelling or grammatical errors
  • References page for citations, linked in the footer
  • Hosted on GitHub and rendered via GitHub Pages
