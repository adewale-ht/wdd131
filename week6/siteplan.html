Project Site Plan

Student: Adekunle Adewale John

Course: WDD 131 – Dynamic Web Fundamentals

Site Name

TechSpark Hub

This name represents a site focused on showcasing interactive and innovative web projects, providing helpful tips, and resources for beginners in web development.

Site Purpose

This site serves as a portfolio and resource hub for aspiring web developers. It provides learning materials, project showcases, and interactive examples to help visitors get started with HTML, CSS, and JavaScript.

Scenarios

  • How can I build a simple interactive website using JavaScript?
  • Where can I view sample projects with source code?

Color Schema

  • Primary Color: #2C3E50 (used for headers and navigation)
  • Accent Color: #18BC9C (used for buttons and links)
  • Background: #f4f4f4 (page background)
  • Text: #333333 (body text)

Typography

  • Montserrat (700): used for all main headings (h1–h3)
  • Roboto Serif: used for body text and paragraphs
